# Dotfiles

Personal configuration for my Arch Linux environment.

## System Overview

- **OS:** Arch Linux
- **WM:** Hyprland (Wayland)
- **Shell:** zsh
- **Terminal:** kitty

- Hyprland + Waybar
- Dunst
- Kitty
- Neovim
- Ranger (rifle/mime)
- Rofi
